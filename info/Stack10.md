# 最大值减去最小值小于或等于num的子数组数量

### 题目

给定数组arr和整数num，共返回多少个子数组满足如下情况：

max(arr[i...j]) - min(arr[i...j]) <= 给定数组arr和整数num

max(arr[i...j])表示子数组arrp[i...j]中的最大值，min(arr[i...j])表示子数组arr[i...j]中的最小值

### 要求

如果数组长度为N，该实现时间复杂度为O(N)的解法。
